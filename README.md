# Hero of the Talk — Redesign Mockup

HTML-mockup av nya heroofthetalk.se. Iteration sker här innan portering till WordPress child theme.

**Live:** https://nahir-prog.github.io/hero-mockup/

## Struktur

- `index.html` — hemsidan
- `assets/styles.css` — design system + components + page styles
- `assets/main.js` — interaktivitet
- `assets/images/` — bilder och ikoner

## Iteration

Öppna `index.html` lokalt eller via GitHub Pages-URL:en ovan. Säg ändringar — vi pushar uppdateringar till repot och GitHub Pages bygger automatiskt om (~1 min).
